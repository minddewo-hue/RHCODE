# 算力趋势研判台

一个离线运行的 A 股算力板块历史趋势研究工具。内置行情为确定性合成演示数据，不代表真实或实时行情，仅供研究，不构成投资建议。

## 启动

```powershell
npm test
npm start
```

默认地址为 `http://127.0.0.1:4178`。可以通过 `PORT` 修改端口。

## CSV 格式

必填列：`date,code,name,close,change_pct,volume,amount`；可选列：`category`。日期格式为 `YYYY-MM-DD`。导入会拒绝无效日期、非数字、缺列、未知列以及重复的 `date + code`。

## 指标口径

- 每只股票以首个有效交易日归一化为 1000，再按交易日等权平均为板块指数。
- 趋势分从 50 分起，按价格与 MA20、MA5/MA20、20 日动量、RSI14、5/20 日量能比和市场宽度加减分。
- 65 分及以上为偏强，39 分及以下为偏弱，其余为震荡。
- 风险规则覆盖量价背离、上涨但宽度下降、20 日回撤超过 8% 和年化波动率超过 35%。

## 限制

工具不处理复权、停牌补齐、成分股历史调整和交易费用。演示数据不应用于任何投资决策；导入数据的准确性与口径由使用者负责。
